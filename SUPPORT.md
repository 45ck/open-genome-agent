# Support

## Questions

- Open a GitHub issue with `[question]` in the title
- Include your operating system, Python version, and the command you ran

## Bug reports

Please include:

- the steps to reproduce
- expected versus actual behavior
- relevant logs or schema validation output
- whether synthetic fixtures or local private data were used

## Security issues

See [SECURITY.md](SECURITY.md). Do not report security issues in public issues.

## Scope note

This project is a research and workflow scaffold. It is not medical software and does not provide clinical support.
