# Public demos

This directory tracks openly shared human genome demos.

Public demos should be:

- openly reusable with clear sharing terms
- reproducible from repo-local instructions
- reported with the same evidence and caveat rules as benchmark runs

The first target is one Harvard Personal Genome Project participant with clear reuse terms and enough public context to produce a readable end-to-end walkthrough.

## Selected sample

The first pinned sample is `huFE0257`.

Why this sample:

- the public profile exposes a whole-genome VCF entry dated `2018-11-26`
- the same profile exposes participant survey and phenotype context
- the profile footer provides an explicit public-reuse statement for the participant genetic and trait dataset

Pinned artifacts:

- sample manifest: `benchmarks/public-demos/huFE0257-manifest.json`
- ingestion guide: `docs/public-demo-ingestion.md`
- privacy boundaries: `docs/public-demo-privacy.md`
- report template: `examples/reports/public_demo_template.md`
- intake scaffold: `scripts/public_demo_walkthrough.py`
