# What We Do Not Claim

- We do not claim diagnostic certainty from this repo.
- We do not claim that benchmark precision and recall equal clinical validity.
- We do not claim that PRS or trait-style outputs belong in the same confidence bucket as direct variant findings.
- We do not claim that a public profile or public genome file removes the need for limitations and caveats.
- We do not claim that functional assay evidence alone settles final clinical significance.
- We do not claim that pharmacogenomics guidance should be followed without human review.
