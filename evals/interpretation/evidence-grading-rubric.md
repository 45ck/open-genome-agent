# Interpretation Evidence-Grading Rubric

Use this rubric when scoring interpretation outputs.

## Source class rules

- direct variant assertions belong to the direct-finding path
- pharmacogenomics guidance belongs to a separate PGx section
- polygenic scores belong to a probabilistic section
- functional assays describe biological effect, not automatic clinical significance

## Confidence mapping

### Strong

Use when the output is tied to:

- a directly observed variant or allele
- a specific source record
- a stronger review-status or guideline-backed interpretation

### Plausible

Use when the result depends on:

- probabilistic scoring
- phenotype translation that still needs context
- functional evidence that is useful but not clinically final

### Hypothesis

Use when the result is only a lead worth follow-up.

### Unsupported

Use when:

- the source does not support the claim
- the source class is wrong for the statement being made
- caveats are missing for a high-impact interpretation

## Failure conditions

Count the case as failed if the output:

- drops the source class
- upgrades uncertainty into certainty
- mixes PRS with direct findings
- turns a guideline into diagnosis or treatment advice
- turns a functional assay result into final clinical truth without caveat
